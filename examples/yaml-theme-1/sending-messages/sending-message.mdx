<h1 className={"openapi__heading"}>Sending Message</h1>

<MethodEndpoint method={"post"} path={"/messages"}></MethodEndpoint>



This endpoint is used to send messages of different types.

<span className={""}><strong>1. Sending Text Message: </strong></span>
Send a simple text message.
<br/>
<br/>

**2. Sending File Message:**
Send a document file.

**3. Sending Image Message:**
Send an image file.

<span className={"deprecated"}><strong>4. Sending List Message: </strong></span>
Send a message containing a list.
<br/>
<br/>

**5. Sending Location Message:**
Send a message with geographical location.


<span className={"deprecated"}><strong>6. Sending Template Message: </strong></span>
Send a message using a predefined template.
<br/>
<br/>
<span className={"deprecated"}><strong>7. Sending Button Message: </strong></span>
Send a message using a predefined template. 
<br/>
<br/>
